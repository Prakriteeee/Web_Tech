Example of Unordered and Ordered Lists

Unordered Lists

Default unordered list:

  • Item 1
  • Item 2
  • Item 3

Unordered list with circle bullets:

  • Item 1
  • Item 2
  • Item 3

Unordered list with square bullets:

  • Item 1
  • Item 2
  • Item 3

Ordered Lists

Default ordered list:

  1. First item
  2. Second item
  3. Third item

Ordered list with lowercase letters:

  1. First item
  2. Second item
  3. Third item

Ordered list with uppercase letters:

  1. First item
  2. Second item
  3. Third item

Ordered list with lowercase Roman numerals:

  1. First item
  2. Second item
  3. Third item

Ordered list with uppercase Roman numerals:

  1. First item
  2. Second item
  3. Third item
